Dependencies:
python 3.14
flask
requests
beautifulsoup4
sqllite3

To install:
clone the repo: https://github.com/jessevfranks/State-Space-Search-for-Software-Auditing
pip install flask requests beautifulsoup4 sqllite3

To run:
In your first terminal from the src/app directory, run:
python victim_app.py

In a second terminal from the src/agent directory, run:
python reporter.py
